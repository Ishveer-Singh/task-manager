# Task Manager

A simple Task Manager built with React that helps users organize and track their daily tasks.

## 🚀 Live Demo

👉 https://ishveer-singh.github.io/task-manager/

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks (All, Done, Not Done)
- Delete all completed tasks
- Task statistics (Total, Done, Pending)
- Local Storage support (tasks persist after page refresh)
- Clean and responsive UI

## Built With

- React
- JavaScript
- CSS
- Vite
